# AI Usage

This project used AI tools including ChatGPT and Codex for planning, scaffolding, code suggestions, debugging, and documentation.

Human contributors made all product, architecture, integration, and final implementation decisions.

AI-assisted areas:

- Product planning and hackathon scope
- Smart contract scaffolding
- Frontend component scaffolding
- ENS/Walrus integration research
- README and demo script drafting

All generated code should be reviewed, edited, and tested by the team before submission.
